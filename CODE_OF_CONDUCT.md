# Code of Conduct

This project adopts the **Contributor Covenant**, version 2.1.

The full text is available at the canonical source:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

In short: be respectful, assume good faith, and help us keep the project a
welcoming place for contributors of every background and experience level.

## Reporting

If you see behaviour that conflicts with the Contributor Covenant, please
contact the maintainer privately at **buluba89@gmail.com**. Reports are
handled confidentially. We respond within a reasonable timeframe and follow
the **Enforcement Guidelines** documented in the Contributor Covenant.

## Scope

The Code of Conduct applies in all project spaces — issues, pull requests,
discussions, commits, and any communication that happens in the project's
name — and in public spaces when an individual is representing the project.
